 CBasicBasic (Base expression) is an expression which contains all other expressions. Its specific behavior is when moving the caret inside the Basic expression. When it encounters a border (last element) on the right side (or the left), the Caret doesn't jump out of the Basic expression and stays within (which makes the Basic expression a parent for all other)
 CCharacterCharacter expression is a terminal expression which holds a single character
 CFractionFraction expression is a TwoExpression which means it consists of two lists of expressions. One list is the numerator and the second one is the denominator

 CMultiplicationSignMultiplicationSign expression is a TerminalExpression pretty much similar to Character, except it has its own template

 CRootRoot expression is a TwoExpression which means it consists of two lists of expressions. One list serves as the degree of the root and the second list serves as the number to calculate the root from
