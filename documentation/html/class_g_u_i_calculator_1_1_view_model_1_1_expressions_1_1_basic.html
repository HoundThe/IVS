Detailed Description

Basic (Base expression) is an expression which contains all other expressions. Its specific behavior is when moving the caret inside the Basic expression. When it encounters a border (last element) on the right side (or the left), the Caret doesn't jump out of the Basic expression and stays within (which makes the Basic expression a parent for all other).
