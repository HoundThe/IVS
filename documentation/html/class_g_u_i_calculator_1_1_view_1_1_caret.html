Detailed Description

A caret is a blinking cursor usually used in textboxes. The Caret class is implemented as a singleton, so only once instance can exist.

The caret is always displayed for a particular expression. Either on the left side of the particular expression or on the right side. The expression by which the caret is displayed is usually being changed, since the caret should be able to move around in the textbox.

In order to move the caret, call the SetActiveExpression method. Sometimes also calling the UpdateActiveExpression method might be needed.

Sometimes it is desired to have more than one instance of the caret class. If such case occurs, a modification of the Caret class will probably be needed.
